- Clone this repository:

  `git clone https://github.com/KingsleyEze/pickup-app.git`

- Getting Started

#### Server

- Enter the `server` folder:

  `cd server`

- Install packages:

  `npm i`

- Run server:

  `npm run start`

#### Client

- Enter the `client` folder:

  `cd server`

- Install packages:

  `npm i`

- Run server:

  `npm run start`
